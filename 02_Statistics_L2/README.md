## 🛠️ 実装プロジェクト: 1.5xIQRによる外れ値検知

### 1. 概要
データの「箱ひげ図（Boxplot）」において、中央の50%のデータ範囲（IQR）から一定以上離れた値を異常値（Outlier）として定義する手法を実装しました。

### 2. 数学的定義
第一四分位数を $Q_1$、第三四分位数を $Q_3$ とし、その差である四分位範囲を $IQR = Q_3 - Q_1$ と定義します。
異常値と判定する境界線（フェンス）は以下の通りです。

$$Lower Bridge = Q_1 - 1.5 \times IQR$$
$$Upper Bridge = Q_3 + 1.5 \times IQR$$

### 3. 解析結果（エビデンス）
Pythonの `seaborn` を用いて、計算によって導き出した境界線（赤点線）と、データの実際の分布を可視化しました。境界線の外側にあるポイントが、統計的に外れ値として処理されます。
